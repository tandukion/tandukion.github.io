Servo Robot Arm

ROS driver for 6DOF Servo Robot Arm based on ROS-Industrial
Servo Robot Arm
Based on library created by ROS-Industrial, it is possible to develop any ROS driver for industrial robot arm.
In this project, I created a ROS driver for a robot arm which uses servo motors as the actuators. This kind of robot is commonly known and one can buy it easily anywhere. The purpose is to make it possible for anyone who wanted to try ROS-Industrial library even on a simple servo robot.
Currently, the ROS driver is defined for 6DOF Servo Robot Arm. The moveit configuration is already set according to the 3D model defined on the robot description.
See repository for ROS Driver
On robot side, ROS controller is defined to handle the communication with the ROS Driver. The ROS Controller includes Simple Message as the main communication protocol from ROS-Industrial. The ROS Controller manages joint commands and robot status message based on that protocol.
See repository for ROS Controller on robot side
Musculoskeletal Robot

A bio-inspired humanoid robot using artificial muscles with agile locomotion skill
Agile Musculoskeletal Robot
We introduced a novel approach in realizing dynamic motion on humanoid robot. Combination of various dynamic motions may lead to better performance of dynamic motion tasks by utilizing the energy transfer between the motions. We have developed a robot capable of doing sequential jumping-stepping motion for agile locomotion.
A bio-inspired robot was developed based on human musculoskeletal system in order to realize human dynamic motion on robot. Pneumatic articial muscle was chosen as the actuator to realize explosive movements.
For realizing the motion, we decided muscle control command for each motion referring to the muscle activation pattern of a human. We have conducted experiments for the sequential jumping-stepping motion with our motion strategy on the developed musculoskeletal robot. The results show that the musculoskeletal robot performs faster stepping by performing pre-jumping before a stepping than stepping without pre-jumping.
The article was published in Proceedings of IEEE International Conference on Robotics and Biomimetics (ROBIO 2018).

RoboSoccer

Faster response and improved performance using Anti-Windup Speed Control System
Anti-Windup Speed Control System for RoboSoccer
RoboCup is a robotics competition which enhance robotics research in an appealing challenge: soccer game. RoboCup Small-size league is one of the league which focuses on multi-agent system in a highly dynamic environment.
Each robot agent has its own low-level controller to handle every command from the server PC. As the environment is highly dynamic, a proper control system needs to be implemented to create an excellent agent. Main challenge for the controller is to have a fast response to speed and direction commands with high frequencies. Hardware limitation becomes another challenge to design the proper controller.
PI anti-windup control system was designed to increase the system response and handle integral windup caused by actuator limitation. The designed controller resulted in improved rise time up to five times which makes the agent agile and suitable with the dynamic environment.
The article was covered in Tokyo Tech Indonesian Commitment Award 2014.

RFID-based Driving License Card

A conceptual design for traffic jam solution
RFID-based Driving License Card
A conceptual design for traffic jam solution.
Jakarta is one of the most congested city in the world. The main problem is the utilization of private vehicles. The government establishes the odd-even number vehicle license plate restriction system in order to solve the problem. However, rule violation keeps taking place where a lot of vehicles which are not supposed to be on the road keep running on the road.
We proposed an idea to increase the effectiveness of the system by utilizing a digital license plate. The license plate may show clearly the state of the vehicle, whether it is on the schedule zone on the day (based on odd-even system).
The system also involves the driver license card for additional features. We proposed an idea to implement RFID on the driver license card. For a driver who gets traffic ticket, the driver license should be updated. With that, license plate can also show the state of the driver, so the cops may know easily for a repeated violation. Also, it makes possible for a case where the system allows a driver to ask for a special permission to drive beside the designated day (e.g odd number car on even date).
The project was done in collaboration between Institut Teknologi Bandung and Kyungpook National University for IS-CAPS (International Students Joint Capstone Design).
Arduino Yun + Android

Interfacing Arduino Yun using an Android program over WiFi
Arduino Yun + Android
Arduino Yun was the first Arduino board with a WiFi module. As this was a new great feature for an Arduino, it was really interesting to try it out. The project was a simple interfacing the Arduino GPIO over WiFi.
In addition, it was also interesting to make the interface on a mobile apps to add more IoT feels. So, I also created an Android apps as the interface to control the Arduino GPIO.
