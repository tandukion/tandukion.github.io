Servo Robot Arm

ROS driver for 6DOF Servo Robot Arm based on ROS-Industrial
Servo Robot Arm
Based on the library created by ROS-Industrial, it is possible to develop any ROS driver for an industrial robot arm.
In this project, I created a ROS driver for a servo robot. This kind of robot is commonly known—one can buy it easily anywhere—that uses servo motors as its actuators. The purpose is to make it possible for anyone wishing to try the ROS-Industrial library, even on a simple servo robot.
Currently, the ROS driver works on a 6DOF Servo Robot Arm. The MoveIt configuration is also already set according to the 3D model defined on the robot description.
See repository for ROS Driver
On the robot side, a ROS controller is designed to handle the communication with the ROS Driver. The ROS Controller includes Simple Message as the main communication protocol from ROS-Industrial. The ROS Controller manages joint commands and robot status messages based on that protocol.
See repository for ROS Controller on robot side
Musculoskeletal Robot

A bio-inspired humanoid robot using artificial muscles with agile locomotion skill
Agile Musculoskeletal Robot
We introduced a novel approach to realizing dynamic motion on a humanoid robot. A combination of various dynamic motions may lead to better performance of dynamic motion tasks by utilizing the energy transfer between the motions. We have developed a robot capable of doing sequential jumping-stepping motion for agile locomotion.
A bio-inspired robot was developed based on the human musculoskeletal system in order to realize human dynamic motion on a robot. Pneumatic artificial muscle was chosen as the actuator to realize explosive movements.
For realizing the motion, we decided muscle control command for each motion referring to the muscle activation pattern of a human. We have conducted experiments for the sequential jumping-stepping motion with our motion strategy on the developed musculoskeletal robot. The results show that the musculoskeletal robot performs faster stepping by performing pre-jumping before a stepping than stepping without pre-jumping.
The article was published in Proceedings of IEEE International Conference on Robotics and Biomimetics (ROBIO 2018).

RoboSoccer

Faster response and improved performance using Anti-Windup Speed Control System
Anti-Windup Speed Control System for RoboSoccer
RoboCup is a robotics competition that enhances robotics research in an appealing challenge: a soccer game. RoboCup Small-size league is one of the leagues which focuses on a multi-agent system in a highly dynamic environment.
Each robot agent has a low-level controller to handle every command from the server PC. As the environment is highly dynamic, an excellent agent needs a proper control system. The main challenge for the controller is to have a fast response to speed and direction commands with high frequencies. Hardware limitation becomes another challenge in designing the proper controller.
A PI anti-windup control system was chosen to increase the system response and handle integral windup caused by actuator limitation. The designed controller improved the rise time up to five times, making the agent agile and suitable for the dynamic environment.
The article was covered in Tokyo Tech Indonesian Commitment Award 2014.

RFID-based Driving License Card

A conceptual design for traffic jam solution
RFID-based Driving License Card
RFID-based Driving License Card is a conceptual design for a traffic jam solution.
Jakarta is one of the most congested cities, with its main problem being the overutilization of private vehicles. The government establishes the odd-even license plate restriction system to solve the problem. However, rule violation keeps taking place where a lot of cars that are not supposed to be on the road keep running on the road.
We proposed an idea to increase the system's effectiveness by utilizing a digital license plate. The license plate shows the car s, whether it is in the scheduled zone on the day (based on the odd-even system).
The system also involves the driver's license card for additional features. We proposed an idea to implement RFID on the driver's license card. By using that, the license plate also shows a driver who gets a traffic ticket. So, the traffic officers may know a repeated violation quickly. Also, it makes it possible for a case where the system allows a driver to ask for special permission to drive beside the designated day (e.g., odd number car on even date).
The project was done in collaboration between Institut Teknologi Bandung and Kyungpook National University for IS-CAPS (International Students Joint Capstone Design).
Arduino Yun + Android

Interfacing Arduino Yun using an Android program over WiFi
Arduino Yun + Android
Arduino Yun was the first Arduino board with a WiFi module. As this was a new great feature for an Arduino at that time, it was interesting to try it out. The project was simple interfacing the Arduino GPIO over WiFi.
In addition, it was also interesting to make the interface from a mobile app to add more IoT feels. So, I also created an Android app as the interface to control the Arduino GPIO.
